# Zoom Virtual Agent Web SDK Mobile Sample

This repository contains sample applications for integrating the Zoom Virtual Agent (ZVA) Web SDK in both iOS and Android environments. Each sample demonstrates how to configure and implement the ZVA Web SDK using platform-specific WebView setups and handle JavaScript callbacks and events.

## Structure

- **ios-sample**: Contains the iOS sample project for integrating the ZVA Web SDK.
- **android-sample**: Contains the Android sample project for integrating the ZVA Web SDK.
